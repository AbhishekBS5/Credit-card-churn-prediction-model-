# Data
This project uses the "Credit Card Customers" dataset from Kaggle (10,127 rows, 20+ features).
Download it here: https://www.kaggle.com/datasets/sakshigoyal7/credit-card-customers
After downloading, place `BankChurners.csv` in this `data/` folder before running
`churn_prediction.py`.
The dataset is not committed to this repo because Kaggle's terms require downloading directly
from the source, and to keep the repo lightweight.
